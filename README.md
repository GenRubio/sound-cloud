
# Sound Cloud

Soun Cloud es una aplicación creada en Electron.js que permite extraer audios de los videos de YouTube y guardarlos en una lista de reproducción.
Tecnologías usadas:
- JQuery
- HTML
- Sass
- Electron.js
## Electron Install
1) En la terminal:
``` bash
npm install
```

3) Para ejecutar el electron escribimos:
``` bash
npm start
```
